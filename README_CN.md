## Features

- 支持版本1.0.0以上(至少现在是这样)
- 已测试适用平台：Win/Ubuntu

## 食用方式

1. `pip install -r requirements.txt`
2. `python typora.py --help`
3. 阅读帮助文档及使用。
4. 做你想做的事。
5. 打包并替换原目录下的 app.asar。 
6. 享受成果。
